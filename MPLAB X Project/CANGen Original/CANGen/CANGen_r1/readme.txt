Configuration list

default
Compile for directly program into Baseboard (without linker).

for_Bootloader
Compile for program into Baseboard via Bootloader (with linker).

debug_with_Bootloader
Compile with Bootloader included in Loadables, you can debug the project with Bootloader programmed.

debug_percepio
You can use this configuration for debugging the project.
This configuration will include Percepio trace analyzer and Bootloader.
